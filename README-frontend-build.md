# Frontend SCSS Build Guide

## Install dependencies

```bash
npm install
```

## Build CSS

```bash
npm run build:css
```

## Build Minified CSS

```bash
npm run build:css:min
```

## Watch for changes (recommended for development)

```bash
npm run watch:css
```

## Output Files

- css/style.css
- css/style.min.css

These files should be included via ag_theme.libraries.yml.
